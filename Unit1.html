Unit 1: JavaScript Basics

This unit introduces the fundamental concepts of JavaScript, covering its syntax, data types, operators, control flow statements, and built-in functions.

1. JavaScript Basics: Syntax

JavaScript code is embedded within HTML documents using the <script> tag. The browser interprets all text between the opening <script> and closing </script> tags as a script.

Script Tag Attributes:

  • language: Specifies the scripting language (e.g., "javascript").
  • type: Specifies the MIME type of the script (e.g., "text/javascript").

Placement of JavaScript:

  • HEAD section: JavaScript placed in the <head> section loads before the <body> content. This is useful when page content depends on the JavaScript code, ensuring the script is loaded first. 
    <html>
<head>
    <title>JavaScript place on head</title>
    <script type="text/javascript">
        document.write("Hello World!");
        document.write("<h2> Hello World! </h2>");
    </script>
</head>
<body>
</body>
</html>
  • BODY section: JavaScript placed in the <body> section loads as the HTML content loads. This can improve page loading speed as content loading is not blocked by JavaScript execution. 
    <html>
<head>
    <title>JavaScript place on body tag</title>
</head>
<body>
    <script type="text/javascript">
        document.write("Hello World!");
    </script>
</body>
</html>
  • Both HEAD and BODY sections: JavaScript can be written in both sections.

2. Comments

Comments are non-executable lines in the code that provide information about the program. They are ignored during execution.

  • Single-line comments: Start with //.
  • Multi-line comments: Enclosed between /* and */.

Example:

<script>
var one = 1; // this is a single line comment
/* this is multi line comment*/
var two = 2;
var three = 3;
</script>

3. JavaScript Data Types

Data types classify the kind of values that can be stored and manipulated in a programming language. JavaScript has two main categories of data types:

Primitive Data Types:

These are predefined and immutable (cannot be changed once created).

  • Number: Represents both integer and floating-point numbers.

    Example:

    var a = 5; // Numeric integer value
var b = 10.5; // Numeric float value
var c = -30.47; // Negative numeric float value
var d = -35; // Negative Numeric integer value
var y = 123e5; // 12300000 (Exponential Notation)
var z = 123e-5; // 0.00123 (Exponential Notation)
  • String: A sequence of characters enclosed in single (') or double (") quotes. Characters are indexed starting from 0.

    Example:

    var name = 'Hello, I am run this town.!'; // Single quote
var name = "Hello, I am run this town.!"; // Double quote
var name = "Hello, I'm run this town.!"; // Single quote use inside string
var name = ""; // an empty string
  • Boolean: Represents logical entities with two values: true or false. Used for logical operations and condition determination.

    Example:

    var val1 = true;
var val2 = false;
  • Null: Indicates the intentional absence of any object value. A variable declared but assigned null means its value is empty.

    Example:

    var str = null; // Value assign null
var car = ""; // The value is "", the type of is "string"
document.write(str == undefined); // Returns true
document.write(null == undefined); // Equality check Returns true
document.write(null === undefined); // Equality with type check Returns false
  • Undefined: A variable that has been declared but has not yet been assigned a value has the value undefined. Its type is also undefined. Uninitialized variables are equal to null in a loose comparison.

    Example:

    var str; // Declare variable without value. Identify as undefined value.
var car; // Value is undefined.
car = undefined; // Value is undefined, type is undefined
  • Symbol: (Not detailed in the provided text, but listed as a primitive type).

Non-primitive (Object) Data Types:

These are reference types, created by the programmer, and are mutable (their values can be changed after creation). They can contain built-in or user-defined objects.

  • Built-in Object Types:
    • Objects
    • Math objects
    • Functions objects
    • Dates objects
    • Arrays objects
    • String objects
  • Creating Objects:

    Syntax:

    var objectname = new Object(); // Create object
var objectname = {}; // Create object
var objectname = new Object(); // Declare object
var objectname = new Object([param1, param2, ..., paramN]); // with parameter

    Examples:

    • Array: 
      var cars = new Array();
var cars = new Array("Saab", "Volvo", "BMW");
var cars = ["Saab", "Volvo", "BMW"];
    • Object Literal: 
      const person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};
    • Date: 
      var d = new Date();
const d = new Date("2022-03-25");
    • String (Object): 
      let x = "John";
let y = new String("John");
    • Math: 
      let x = Math.PI;
let y = Math.sqrt(16);

4. JavaScript Variables

Variables are containers for storing data. They must be identified with unique names called identifiers.

Naming Rules for Variables:

  1. Can contain letters (A-Z, a-z), digits (0-9), underscores (_), and dollar signs ($).
  2. Must begin with a letter, underscore, or dollar sign. Cannot start with a number.
  3. Are case-sensitive (y and Y are different variables).
  4. Cannot be JavaScript reserved keywords (e.g., if, switch, else, for, var, let, const).

Example (Invalid Names):

var _abc = "Hi!";
var abc = "Hello!";
var 9abc = "Bye!"; // This is invalid
var for = "9"; // This is invalid (reserved keyword)

Declaring Variables:

  • Automatically: 
    x = 5; // automatically declared
  • Using var keyword: (Function-scoped, can be re-declared and re-assigned) 
    var y = 6; // declared Using var keyword
  • Using let keyword: (Block-scoped, can be re-assigned but not re-declared in the same scope) 
    let z = 12; // declared Using let keyword
  • Using const keyword: (Block-scoped, cannot be re-assigned or re-declared) 
    const a = 3.14; // declared Using const keyword

Variable Initialization:

Storing a value in a variable using the assignment operator (=). Multiple variables can be declared and initialized in a single statement, separated by commas.

  • Syntax: 
    var variable_name = value;
var var1 = val1, var2 = val2, ..., varN = valN;
  • Example: 
    var num = 5;
var num1 = 5, num2 = 6, num3 = 7;

Variable Scope:

Determines where a variable can be accessed.

  • Global Scope: Variables declared outside any function. Accessible from anywhere in the code (inside and outside functions).

    Example:

    <script>
var data = 200; // global variable
function a() {
    document.write(data);
}
function b() {
    document.write(data);
}
a(); // calling JavaScript function
b();
</script>
  • Local Scope: Variables declared inside a block or function. Accessible only within that block or function.

    Example:

    <script>
var myVar = "global"; // Declare a global variable
function abc() {
    var myVar = "local"; // Declare a local variable
    document.write(myVar);
}
</script>

5. JavaScript Literals

Literals represent fixed values that cannot be changed.

  • Numeric Literal: Integer values (decimal, hexadecimal, octal).

    Example:

    120; // decimal literal
021434; // octal literal
0x4507; // hexadecimal literal
  • Floating-point Literal: Numbers with a decimal point, possibly with an exponent.

    Example:

    6.99689; // floating-point literal
-167.39894; // negative floating-point literal
  • Boolean Literal: true or false.

    Example:

    true; // Boolean literal
false; // Boolean literal
  • String Literal: Characters enclosed in single or double quotes.

    Example:

    "Study"; // String literal
'tonight'; // String literal
  • Array Literal: An array created using square brackets [] with or without values.

    Example:

    ["Abhishek", "Supriya", "Joey"]; // Array literal
let students = ["Abhishek", "Supriya", "Joey"]; // Array literal
  • Object Literal: A collection of key-value pairs enclosed in curly braces {}.

    Example:

    var games = {cricket: 11, chess: 2, carom: 4}; // Object literal

6. JavaScript Type Conversion (Casting)

The process of converting data from one data type to another.

  • Implicit Conversion (Automatic Type Conversion): Done automatically by JavaScript.

    Example:

    When using the + operator with a string and a number, the number is converted to a string, and concatenation occurs.

    <script>
document.write("100" + 24 + "<br />"); // Output: 10024
document.write('100' + false + "<br />"); // Output: 100false
document.write("100" + null + "<br />"); // Output: 100null
document.write("100" + undefined + "<br />"); // Output: 100undefined
</script>
  • Explicit Conversion (Manual Type Conversion): Done manually by the programmer using built-in functions.

    Example:

    <script>
document.write(Number("200") + "<br />"); // Output: 200
document.write(Number("100AB-2") + "<br />"); // Output: NaN
document.write(Number(false) + "<br />"); // Output: 0
document.write(Number(null) + "<br />"); // Output: 0
document.write(Number(undefined) + "<br />"); // Output: NaN
document.write(+"200" + "<br />"); // Output: 200 (Unary plus operator for conversion)
</script>

7. JavaScript Operators

Operators are special symbols that perform operations on one or more operands.

  • Arithmetic Operators: Perform mathematical calculations.

    + (Addition), - (Subtraction), * (Multiplication), / (Division), % (Modulus), ++ (Increment), -- (Decrement).

    Examples:

    var x = 10, y = 5;
document.write(x + y); // RESULT: 15
document.write(x - y); // RESULT: 5
document.write(x * y); // RESULT: 50
document.write(x / y); // RESULT: 2
document.write(x % y); // RESULT: 0
// Increment/Decrement
var x = 10;
document.write(x++); // POST INCREMENT: x: 10, after increment x become now 11
document.write(++x); // PRE INCREMENT: x: 11, before increment x become now 11
document.write(x--); // POST DECREMENT: x: 10, after decrement x become now 9
document.write(--x); // PRE DECREMENT: x: 9, before decrement x become now 9
  • Assignment Operators: Assign values to variables.

    = (Simple Assignment), += (Add and Assignment), -= (Subtract and Assignment), *= (Multiply and Assignment), /= (Divide and Assignment), %= (Modulus and Assignment).

    Examples:

    var y = 10; var x = y; // x=10
var x = 10, y = 5;
x += y; // x=15
x -= y; // x=5
x *= y; // x=50
x /= y; // x=2
x %= y; // x=0
  • Comparison Operators: Compare two values and return a boolean result (true or false).

    == (Equal), != (Not Equal), > (Greater than), < (Less than), >= (Greater than or Equal to), <= (Less than or Equal to).

    Examples:

    var a = 5, b = 5;
document.write(a == b); // OUTPUT: true
var a = 5, b = 10;
document.write(a != b); // OUTPUT: true
document.write(a > b); // OUTPUT: false
document.write(a < b); // OUTPUT: true
var a = 5, b = 5;
document.write(a >= b); // OUTPUT: true
document.write(a <= b); // OUTPUT: true
  • Logical Operators: Perform logical operations on Boolean values, combining conditions.

    && (Logical AND), || (Logical OR), ! (Logical NOT).

    Examples:

    var a = 5, b = 10;
document.write((a != b) && (a < b)); // OUTPUT: returns true
document.write((a > b) || (a == b)); // OUTPUT: returns false
document.write((a < b) || (a == b)); // OUTPUT: returns true
document.write(!(a < b)); // OUTPUT: returns false
document.write(!(a > b)); // OUTPUT: returns true
  • Conditional Operator (Ternary Operator): A shorthand for an if-else statement.

    Syntax:

    condition ? value1 : value2;

    Example:

    var a = 10, b = 5;
var c = a > b ? a : b; // value of c would be 10
var d = a > b ? b : a; // value of d would be 5
  • String Concatenation: The + operator performs concatenation when one of the operands is a string.

    Example:

    let a = 5, b = "Hello", c = "World! ", d = 10;
document.write(a + b); // returns "5Hello"
document.write(b + c); // returns "Hello World!"
document.write(a + d); // returns 15
document.write(b + true); // returns "Hello true"
document.write(c - b); // returns NaN; - operator can only use with numbers

8. User Interaction through Dialog Boxes (Popup Boxes)

JavaScript provides built-in global functions to display popup message boxes.

  • alert(message): Displays a popup box with a specified message and an "OK" button. Code execution pauses until "OK" is clicked.

    Syntax:

    alert("your message here");

    Example:

    <script>
function showAlert() {
    alert("One click event on button");
}
</script>
<button onclick="showAlert()">Click for Alert</button>
  • prompt(message, defaultValue): Displays a popup box with a message, an input field, and "OK" and "Cancel" buttons. Returns the user's input (or null if "Cancel" is clicked).

    Syntax:

    prompt("your message", "default value");

    Example:

    <script>
function showPrompt() {
    var a;
    a = prompt("Enter your Name", "");
    if (a !== null) { // Check if user didn't cancel
        alert("Hello " + a);
    } else {
        alert("Prompt cancelled.");
    }
}
</script>
<button onclick="showPrompt()">Click for Prompt</button>
  • confirm(message): Displays a popup box with a message and "OK" and "Cancel" buttons. Returns true if "OK" is clicked, false if "Cancel" is clicked.

    Syntax:

    confirm("your message");

    Example:

    <script>
function showConfirm() {
    var r = confirm("Press a button");
    if (r == true) {
        alert("You pressed OK!");
    } else {
        alert("You pressed Cancel!");
    }
}
</script>
<button onclick="showConfirm()">Click for Confirm</button>

9. Built-in Functions

  • parseInt(string, radix): Parses a string argument and returns an integer. radix specifies the base of the number (2 to 36).

    Example:

    <script>
document.write("parseInt('10', 10): " + parseInt("10", 10) + "<br>"); // Output: 10
document.write("parseInt('10.00'): " + parseInt("10.00") + "<br>"); // Output: 10
document.write("parseInt('10.33'): " + parseInt("10.33") + "<br>"); // Output: 10
document.write("parseInt('34 45 66'): " + parseInt("34 45 66") + "<br>"); // Output: 34
document.write("parseInt('60'): " + parseInt("60") + "<br>"); // Output: 60
document.write("parseInt('40 years'): " + parseInt("40 years") + "<br>"); // Output: 40
document.write("parseInt('He was 40'): " + parseInt("He was 40") + "<br>"); // Output: NaN
</script>
  • parseFloat(value): Parses a string argument and returns a floating-point number. Ignores leading/trailing whitespace. Returns NaN if the string doesn't start with a valid number.

    Example:

    <script>
document.write("parseFloat('40.00'): " + parseFloat("40.00") + "<br>"); // Output: 40
document.write("parseFloat('40'): " + parseFloat("40") + "<br>"); // Output: 40
document.write("parseFloat('40 years'): " + parseFloat("40 years") + "<br>"); // Output: 40
document.write("parseFloat('40 H'): " + parseFloat("40 H") + "<br>"); // Output: 40
document.write("parseFloat('H40'): " + parseFloat("H40") + "<br>"); // Output: NaN
</script>
  • isNaN(value): Checks whether a given value is NaN (Not a Number). Returns true if the value is NaN, false otherwise.

    Example:

    <script>
var value1 = 123;
var value2 = NaN;
document.write("Value1 is " + isNaN(value1) + "<br>"); // Output: Value1 is false
document.write("Value2 is " + isNaN(value2) + "<br>"); // Output: Value2 is true
</script>
  • Number(value): Converts a variable or value into a number. Returns a primitive number value.

    Example:

    <script>
var x = 3.14;
var y = 3;
var z = "ABC";
document.write("the value of x: " + Number(x) + "<br>"); // Output: the value of x: 3.14
document.write("the value of y: " + Number(y) + "<br>"); // Output: the value of y: 3
document.write("the value of z: " + Number(z) + "<br>"); // Output: the value of z: NaN
</script>

10. Flow Control Statements: Decision-Making and Looping

Control flow statements determine the order in which instructions are executed.

Decision-Making (Conditional Statements):

Perform different actions based on different conditions.

  • if Statement: Executes a block of code if a specified expression is true.

    Syntax:

    if (expression) {
    // Statement(s) to be executed if expression is true
}

    Example:

    <script>
if (1 > 0) {
    document.write("1 is greater than 0<br>");
}
if (1 < 0) {
    document.write("1 is less than 0<br>"); // This will not be executed
}
</script>
  • if...else Statement: Executes one block of code if the expression is true, and another block if it's false.

    Syntax:

    if (expression) {
    // Statement(s) to be executed if expression is true
} else {
    // Statement(s) to be executed if expression is false
}

    Example:

    <script>
var mySal = 500;
var yourSal = 1000;
if (mySal > yourSal) {
    document.write("My Salary is greater than your salary<br>");
} else {
    document.write("My Salary is less than or equal to your salary<br>");
}
</script>
  • else if... Statement (Ladder): Allows checking multiple conditions sequentially.

    Syntax:

    if (expression1) {
    // Statement(s) if expression1 is true
} else if (expression2) {
    // Statement(s) if expression2 is true
} else if (expression3) {
    // Statement(s) if expression3 is true
} else {
    // Statement(s) if no expression is true
}

    Example:

    <script>
var a = 20;
if (a == 10) {
    document.write("a is equal to 10<br>");
} else if (a == 15) {
    document.write("a is equal to 15<br>");
} else if (a == 20) {
    document.write("a is equal to 20<br>");
} else {
    document.write("a is not equal to 10, 15 or 20<br>");
}
</script>
  • switch Statement: Executes different blocks of code based on the value of an expression.
    • break: Terminates the switch statement and continues execution after it.
    • default: Specifies the code to execute if none of the case labels match the expression's value.

    Syntax:

    switch (expression) {
    case condition1:
        statement(s);
        break;
    case condition2:
        statement(s);
        break;
    // ...
    default:
        statement(s);
}

    Example:

    <script>
var weekname = "wednesday";
document.write("Today is: ");
switch (weekname) {
    case "monday":
        document.writeln("Monday working day.<br>");
        break;
    case "tuesday":
        document.writeln("Tuesday working day.<br>");
        break;
    case "wednesday":
        document.writeln("Wednesday government holiday.<br>");
        break;
    case "thursday":
        document.writeln("Thursday working day.<br>");
        break;
    case "friday":
        document.writeln("Friday is a last working day of the week.<br>");
        break;
    case "saturday":
        document.writeln("Saturday week end.<br>");
        break;
    case "sunday":
        document.writeln("Sunday week end.<br>");
        break;
    default:
        document.writeln("No any case found.<br>");
}
</script>
  • break Statement (in loops): Terminates the current loop and transfers execution to the statement immediately following the loop.

    Syntax:

    break; or break label;

    Example:

    <script>
var i = 0;
document.write("Break Example: ");
while (i <= 10) {
    if (i == 5) {
        document.write("terminated, i = 5<br>");
        break;
    }
    document.write(i + " ");
    i++;
}
</script>
<!-- Expected Output: Break Example: 0 1 2 3 4 terminated, i = 5 -->
  • continue Statement: Skips the current iteration of a loop and proceeds to the next iteration.

    Syntax:

    continue;

    Example:

    <script>
var i = 0;
document.write("Continue Example: ");
while (i < 10) {
    i++;
    if (i == 5) {
        document.write("skipped, i = 5 ");
        continue;
    }
    document.write(i + " ");
}
</script>
<!-- Expected Output: Continue Example: 1 2 3 4 skipped, i = 5 6 7 8 9 10 -->

Looping Statements:

Execute a block of code repeatedly.

  • for Loop: Executes a block of code a specific number of times, based on initialization, condition, and increment/decrement.

    Syntax:

    for (initialization; condition; increment/decrement) { statements; }

    Example:

    <script>
for (var num = 1; num <= 3; num++) {
    document.write(num + ": iteration<br>");
}
document.write("End of for loop<br>");
</script>
<!-- Expected Output:
1: iteration
2: iteration
3: iteration
End of for loop
-->
  • while Loop: Executes a block of code as long as a specified condition is true. The condition is checked before each iteration.

    Syntax:

    while (condition) { statements; increment; }

    Example:

    <script>
var number = 1;
document.write("While Loop Example:<br>");
while (number <= 3) {
    document.writeln(number + " times<br>");
    number++;
}
</script>
<!-- Expected Output:
While Loop Example:
1 times
2 times
3 times
-->
  • do...while Loop: Executes a block of code at least once, and then repeatedly as long as a specified condition is true. The condition is checked after each iteration.

    Syntax:

    do { statements; increment; } while (condition);

    Example:

    <script>
var number = 1;
document.write("Do-While Loop Example:<br>");
do {
    document.writeln(number + " times<br>");
    number++;
} while (number <= 3);
</script>
<!-- Expected Output:
Do-While Loop Example:
1 times
2 times
3 times
-->

11. Array

The JavaScript Array object allows storing multiple values in a single variable. Arrays can hold various data types and are accessed by numeric indices starting from 0.

  • Creating a JavaScript Array:
    • Array Literal Method:

      Syntax:

      my_array = [value1, value2, ..., valueN];

      Example:

      var numArr = [10, 20, 30, 40, 50];
var domain = [".com", ".net", ".org", ".gov"];
    • Built-in Array Constructor Method:

      Syntax:

      my_array = new Array(value1, value2, ..., valueN); or my_array = new Array(array_size);

      Example:

      var numArr = new Array(10, 20, 30, 40, 50);
var domain = new Array(".com", ".net", ".org", ".gov");
var emptyArray = new Array(10); // an empty array of length 10
  • Accessing Array Elements: Elements are accessed using their index in square brackets.

    Syntax:

    arrayName[index]

    Example:

    <script type="text/javascript">
// Create empty array
var domain = new Array();
// Initializing (Assigning) values
domain[0] = ".com";
domain[1] = ".net";
domain[2] = ".org";
domain[3] = ".gov";
// Accessing Array Elements
document.write("Domain 0: " + domain[0] + "<br>");
document.write("Domain 1: " + domain[1] + "<br>");
document.write("Domain 2: " + domain[2] + "<br>");
document.write("Domain 3: " + domain[3] + "<br>");
</script>
<!-- Expected Output:
Domain 0: .com
Domain 1: .net
Domain 2: .org
Domain 3: .gov
-->

12. User-defined Functions

Functions are reusable blocks of code that perform a specific task. They help in organizing code and avoiding repetition.

  • Function Declaration/Definition:

    Syntax:

    function functionName(arguments/parameter) {
    statements
}

    Example:

    <script type="text/javascript">
function msgfun() { // function definition
    document.write("Hello world from msgfun!<br>"); // statements
}
msgfun(); // calling the function
</script>
  • Calling a Function: Executing the function by its name followed by parentheses ().

    Example:

    <script>
function greet() {
    document.write("Hello World from greet!<br>");
}
greet(); // calling function
</script>
  • Function Parameters (Arguments): Variables listed inside the parentheses in the function definition, used to pass values into the function.

    Syntax:

    function functionName(parameter1, parameter2, parameter3) {
    // statements
}

    Example:

    <script>
function greetWithNames(firstName, lastName) {
    document.write("Hello " + firstName + " " + lastName + "!<br>");
}
greetWithNames("Bill", "Gates"); // values will be passed to function
</script>
<!-- Expected Output: Hello Bill Gates! -->
  • Return a Value From Function: A function can send a value back to the calling code using the return keyword.

    Example:

    <script>
var add = function (num1, num2) {
    return num1 + num2; // Return a value
};
var result = add(10, 20); // values will be passed
document.write("Result of addition: " + result + "<br>");
</script>
<!-- Expected Output: Result of addition: 30 -->

Note: Some JavaScript examples use document.write() for simplicity. In modern web development, it's generally recommended to manipulate the DOM (Document Object Model) using methods like document.getElementById().innerHTML = ... or document.createElement() for dynamic content updates.
